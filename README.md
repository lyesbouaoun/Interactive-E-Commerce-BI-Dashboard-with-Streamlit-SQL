# Interactive E-Commerce BI Dashboard with Streamlit & SQL

Interactive Business Intelligence dashboard built with Python, Streamlit and SQLite for analyzing an e-commerce dataset.

## Features

* KPI monitoring (sales, revenue, average order value)
* Product category analysis
* Sales and revenue trends by month
* Payment transaction analysis
* Dynamic filters by city, category and payment type
* Interactive charts and pivot tables

## Technologies

* Python
* Streamlit
* SQLite
* Pandas
* Matplotlib

## Dataset

Brazilian E-Commerce Public Dataset by Olist.

## Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
